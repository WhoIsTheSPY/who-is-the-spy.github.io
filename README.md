# Who Is the Spy

A polished pass-and-play social deduction game for 4–7 players.

## Play in a browser

Open `index.html` in any modern browser. The experience includes:

- Animated splash logo page
- Menu page with player count, friend/AI mode, and AI difficulty
- Game page with private word reveal, discussion timer, spy vote, and results

## Android version

The `android` folder contains a minimal Android WebView wrapper that loads the game from `android/app/src/main/assets/index.html`.

### How to open

1. Open Android Studio.
2. Choose `Open` and select the `android` folder.
3. Build and run on an emulator or device.

The Android app is written in Java and uses a WebView to display the HTML/CSS/JS game.
